API Project: URL Shortener Microservice

User Story:

  1. Via the form below, I can POST a URL to https://robbyxps-fcc-manage-packages-03-4.glitch.me/api/shorturl/new and I will receive a shortened URL in the JSON response.
    Example : {"original_url":"https://www.freecodecamp.org","short_url":20200}
  2. If I pass an invalid URL such as www.freecodecamp, the JSON response will contain an error like {"error":"invalid URL"}
  3. When I visit the shortened URL, it will redirect me to my original link.
    Example : https://robbyxps-fcc-manage-packages-03-4.glitch.me/api/shorturl/20200
    will redirect to https://www.freecodecamp.org/
